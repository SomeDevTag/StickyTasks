# StickyTasks

A lightweight, freeform collaborative task board for local networks.

Create sticky notes, drag them anywhere, organize them into pages, connect related tasks, and collaborate live with others on the same network. Everything is stored locally in a simple `tasks.json` file.

## Features

- 📝 Freeform sticky note board
- 🎨 Color-coded tasks
- 📂 Multiple pages/workspaces
- 🔗 Link related tasks together
- ✅ Drag tasks into the completion zone
- 🌐 Real-time collaboration over your local network
- 💾 Automatic local saving (`tasks.json`)
- 🌙 Dark mode
- 📤 Export & Import boards

## Getting Started

1. Install Python.
2. Run `Start StickyTasks.bat`.
3. Your browser will open automatically.
4. Share the LAN address with others on the same network to collaborate.

## Data Storage

All tasks are stored locally in:
tasks.json

If the server isn't running, opening `task-manager.html` directly will use browser storage instead.
